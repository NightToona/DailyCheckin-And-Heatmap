# DailyCheck-In And Heatmap 1.0.0
DailyCheckin-And-Heatmap，

一个简约易用的 **每日打卡(日历视图)**、**内容热力图** 与 **统计数据** 的一体化插件，专为 Obsidian！

让打卡更加轻便，更加专注于笔记内容。

## 界面视图

界面一共包含三部分：
- 每日打卡
- 每日内容热立图
- 数据统计

### 📆每日打卡

- 通过点击顶部打卡按钮，实现日历打卡记录；
- 点击对应打卡日期，可手动取消选中日期打卡记录；

### 🔥每日内容热力图

- 每日热力图更新根据前一天字数变化量更新强度；

### 📊数据统计

- 包含累计打卡天数、累计满周数 和 连续天数；

（此处的清空统计只作用与该部分的数据统计，不会影响第一、二部分数据）

## 为 Obsidian 安装的方法

将文件夹 `daily-checkin-heatmap` 内容完整下载，在对应的 Obsidian 中的仓库找到 `仓库\.obsidian\plugins`文件夹，并将下载内容完整放入该路径中即可使用。（记得重启Obsidian）
